# declarationfiscale_projet
Projet de gestion de la déclaration fiscale des clients pour un cabinet comptable

## Installation des dépendances (Pipfile)

1. Installer pipenv: `pip install --user pipenv` (si pas encore installé)
2. Se placer à la racine du projet (Pipfile présent).
3. Installer dépendances: `pipenv install

### Commandes optionnelles
4. Activer l'environnement: `pipenv shell`
5. Ajouter (installer) une dépendance:
   - Prod: `pipenv install <package>`
6. Mettre à jour: `pipenv update <package>` (ou sans package pour tout).
7. Exporter requirements.txt (optionnel): `pipenv requirements > requirements.txt`
8. Supprimer l'environnement virtuel: `pipenv --rm`
